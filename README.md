# Audiculator

Audiculator is a web-based 3D audio visualizer

[Live Demo](https://audiculator.berikai.dev)

![Preview GIF](./preview.gif)

## ⚡️ Features

- ☀️ Simple & Elegant

- ⚙️ Customizable

- 🎚️ Preset Support

- 📷 Video Export

- 📍 Keyframe Editor (not yet)
